External JavaScript file

We can create external JavaScript file and embed it in many html page.It provides code re usability because single JavaScript file can be used in several html pages.
An external JavaScript file must be saved by .js extension. It is recommended to embed all JavaScript files into a single file. It increases the speed of the webpage.

An external javascript file is usally added with .js extension before the closing </body> tag.
There is another powerful method for generating output by using console.log() function.
